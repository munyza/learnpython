# Variables 
 
## Explanation 
Variables are used to store data in Python. 
Each variable has a **data type**, such as: 
- 'int' (integer)
- 'float' (decimal)
- 'str' (string)
- 'bool' (True or False)

Variables must start with a letter or underscore and cannot be Python keywords. 

### Examples 
```python
name = "Alice"
age = 25
height = 5.6
is_student = True
